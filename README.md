## EverestWather

# Switf Weather iOS Source Code

## Installation

```bash
$ [sudo] gem install cocoapods
```

`cd /Users/someone/Documents/Everest\ Wather`

```bash
$ pod init
```

## Open Podfile
Open `Podfile`, and add your dependencies.

`pod 'SwiftyJSON'`

`pod 'Alamofire'`

`pod 'SVProgressHUD'`

```bash
Save your Podfile.
```

```bash
$ pod install
```
